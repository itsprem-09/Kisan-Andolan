# Rashtriya Kishan Manch

This project is a MERN stack application for Rashtriya Kishan Manch, designed to provide information, engage users, and manage organizational content.

## Project Overview

- **Technology Stack**: MongoDB, Express.js, React.js, Node.js
- **Key Features**: Informational pages, member registration, program showcases, admin panel for content management.

## Setup Instructions

Detailed setup instructions will be added as the project progresses.

### Backend Setup

1.  Navigate to the `server` directory: `cd server`
2.  Install dependencies: `npm install`
3.  Create a `.env` file based on `.env.example` (to be created) and add your MongoDB URI, JWT Secret, etc.
4.  Start the server: `npm start` (or `npm run dev` for development with nodemon).

### Frontend Setup

1.  Navigate to the `client` directory: `cd client`
2.  Install dependencies: `npm install`
3.  Start the client: `npm start`
